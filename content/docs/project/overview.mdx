# Project Overview

Dokumen ini mencakup struktur pelaksanaan, alur kerja, serta KPI yang berlaku untuk seluruh jenis project pada ekosistem YYZU, termasuk namun tidak terbatas pada Mini Project dan Collaborative Project.

---

## 1. Struktur Pelaksanaan Project

| Level | Peran | Fungsi |
| :--- | :--- | :--- |
| **1** | Mentor (Instructor + Client) | Memberikan brief dan arahan teknis |
| **2** | Divisi Product & Project Management | Menerjemahkan brief menjadi draf scope dan timeline, mendistribusikan ke PIC |
| **3** | PIC / Ketua Kelompok | Melakukan koordinasi harian dalam kelompok |
| **4** | Anggota Kelompok | Melaksanakan task sesuai pembagian PIC |

Divisi Product & Project Management mengoordinasikan seluruh PIC yang berjalan secara paralel dalam satu batch. PIC merupakan satu-satunya jalur komunikasi resmi antara kelompok dan Divisi Product & Project Management.
