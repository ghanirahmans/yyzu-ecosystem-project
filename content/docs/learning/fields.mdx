# Bidang Keahlian (Fields of Expertise)

Ekosistem YYZU membagi area fokus pembelajaran dan kolaborasi ke dalam beberapa bidang keahlian digital utama yang relevan dengan kebutuhan industri digital modern:

---

## 1. Software & Web Development
Pembelajaran pengembangan aplikasi berbasis web menggunakan teknologi modern.
- **Topik**: HTML/CSS, JavaScript/TypeScript, React/Next.js, Node.js, RESTful API, database management, dan integrasi frontend-backend.

## 2. Mobile Development
Pengembangan aplikasi gawai (*mobile apps*) lintas platform maupun asli.
- **Topik**: Flutter, React Native, Swift (iOS), Kotlin (Android), mobile UX guidelines, dan manajemen status aplikasi.

## 3. UI/UX Design
Riset pengguna, pembuatan arsitektur informasi, serta perancangan antarmuka visual produk.
- **Topik**: User Research, Wireframing, High-Fidelity UI Design, Prototyping, Figma Design System, Usability Testing, dan UX Writing.

## 4. Product Management
Pengelolaan siklus produk digital dari tahap ideasi, perencanaan fitur, hingga peluncuran.
- **Topik**: Product Thinking, Requirement Gathering (PRD), User Stories, Roadmap Planning, Prioritization Frameworks (RICE), dan Product Analytics.

## 5. Quality Assurance (QA)
Pengujian kualitas, keandalan, keamanan, dan kinerja aplikasi sebelum rilis.
- **Topik**: Manual Testing, Automation Testing, Test Case Writing, Bug Reporting, API Testing, dan Regression Testing.

## 6. Data & AI (Kecerdasan Buatan)
Pengolahan data, analisis informasi, serta penerapan model pembelajaran mesin.
- **Topik**: Python, SQL, Data Analysis, Machine Learning, Data Visualization, dan prompt engineering untuk solusi AI.

## 7. DevOps & Cloud
Manajemen infrastruktur server, otomasi penyebaran aplikasi, dan skalabilitas sistem.
- **Topik**: CI/CD pipelines, Docker, Cloud Services (AWS/GCP), Linux Admin, SSL, Domain Management, dan server monitoring.

## 8. Creative Technology & Digital Product
Eksplorasi teknologi kreatif untuk menghasilkan solusi produk digital yang bernilai bisnis.
- **Topik**: Landing Page Optimization, Content Management Systems (CMS), No-Code/Low-Code Tools, dan Web Animation.
