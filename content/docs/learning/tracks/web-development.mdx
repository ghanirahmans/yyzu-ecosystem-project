# Track Web Development

Track Web Development di YYZU dirancang untuk membantumu menjadi web developer yang siap kerja. Kamu akan belajar membangun website dan aplikasi web dari nol sampai siap dipakai.

---

## Apa yang Akan Dipelajari?

### Fase Beginner

- **HTML & CSS** - fondasi setiap website
- **JavaScript dasar** - membuat halaman jadi interaktif
- **Git & GitHub** - version control untuk kolaborasi
- **Responsive design** - website yang bagus di semua device
- **Dev tools** - browser developer tools, terminal dasar

### Fase Intermediate

- **JavaScript lanjutan** - async, API, DOM manipulation
- **Framework** - React, Next.js, atau yang relevan
- **Backend dasar** - Node.js, API, database
- **Testing** - unit test dan integration test
- **Deployment** - cara mempublish website

### Fase Advanced

- **Fullstack development** - integrasi frontend dan backend
- **Performance optimization** - membuat website cepat
- **Security** - best practices keamanan web
- **Architecture** - struktur aplikasi yang scalable
- **DevOps dasar** - CI/CD, containerization

### Peer Leader

- **Tech lead** - memimpin keputusan teknis
- **Code review** - memastikan kualitas kode tim
- **Mentoring** - membimbing talenta baru
- **Kurikulum** - berkontribusi dalam pengembangan materi

---

## Mini Project Contoh

| Project | Level | Deskripsi |
|---------|-------|-----------|
| Landing page | Beginner | Website satu halaman dengan HTML/CSS |
| Todo app | Beginner-Intermediate | Aplikasi to-do list dengan JavaScript |
| Weather app | Intermediate | Aplikasi yang mengambil data dari API |
| Portfolio website | Intermediate | Website pribadi untuk menampilkan karya |
| E-commerce MVP | Advanced | Aplikasi belanja sederhana fullstack |

## Kolaborasi dengan Track Lain

Dalam collaborative project, kamu akan bekerja sama dengan:

- **UI/UX Designer** - mereka merancang, kamu implementasi
- **Product Manager** - mereka mengelola prioritas, kamu bangun fitur

Ini meniru dinamika kerja di perusahaan teknologi sungguhan.

---

## Siapa yang Cocok?

Track ini cocok untuk kamu yang:

- ✅ Suka memecahkan masalah dengan kode
- ✅ Penasaran bagaimana website dibuat
- ✅ Ingin melihat hasil kerja secara langsung
- ✅ Tertarik dengan teknologi web terbaru
