# Software Development

Track ini membangun kemampuan rekayasa perangkat lunak dari fondasi hingga arsitektur sistem. Kamu akan belajar pemrograman, pola desain, pengujian, dan manajemen teknis. Cocok untuk yang ingin jadi Software Engineer, Backend Developer, atau Full-Stack Developer.

**Kompetensi per Level:**

- Beginner: 5 kompetensi
- Intermediate: 4 kompetensi
- Advanced: 3 kompetensi

## Level: Beginner

### Programming Fundamentals & Logic

- **Mastery Level:** Level 1 - Understand & Articulate
- **Evidence:** Kode program fungsional sederhana
- **Exit Criteria:** Lolos diagnostic questions logika dasar tanpa error sintaks

### Algoritma & Struktur Data

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Implementasi array, map, atau set dalam kode
- **Exit Criteria:** Memilih struktur data yang tepat untuk menyelesaikan masalah efisiensi memori

### Object-Oriented Programming (OOP) & FP

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Struktur class, interface, atau functional pipeline
- **Exit Criteria:** Kode modular tanpa inheritance berlebihan

### Clean Code & Naming Convention

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Kode yang mudah dibaca oleh developer lain
- **Exit Criteria:** Mengikuti naming convention track dan bebas dari magic numbers

### Version Control Praktis (Git)

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Repositori git dengan commits rapi dan PR
- **Exit Criteria:** Branching dan merge request teratur tanpa konflik besar

## Level: Intermediate

### Design Patterns (GoF)

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Kode menggunakan Pattern (misal: Singleton, Factory, Observer)
- **Exit Criteria:** Mampu menjelaskan trade-off pattern yang diimplementasikan

### SOLID Principles

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Refactoring kode yang melanggar SRP atau OCP
- **Exit Criteria:** Class dan module mudah diperluas tanpa memodifikasi kode lama

### Testing Strategy (Unit, Integration, E2E)

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** File test otomatis (Jest, PyTest, dll.) dengan test coverage memadai
- **Exit Criteria:** Menguji happy path dan edge case fungsi kritis

### Dokumentasi Teknis & API Contract

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Dokumentasi OpenAPI/Swagger atau schema contract
- **Exit Criteria:** Konsumen API (frontend/mobile) dapat beroperasi tanpa klarifikasi verbal

## Level: Advanced

### System Design & Scalability

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Dokumen arsitektur sistem (ADR) dan skema sistem terdistribusi
- **Exit Criteria:** Sistem dirancang mampu menangani beban tinggi secara teoritis dan teruji

### Code Review Leadership

- **Mastery Level:** Level 4 - Peer Instruction & Knowledge Share
- **Evidence:** Tinjauan PR yang konstruktif di GitHub
- **Exit Criteria:** Memberikan feedback edukatif berbasis SBIN yang membantu mentee berkembang

### Technical Debt Management

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Rencana refactoring terukur dan eksekusi pengurangan utang teknis
- **Exit Criteria:** Kualitas codebase meningkat tanpa menghentikan rilis fitur baru
