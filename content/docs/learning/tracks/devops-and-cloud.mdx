# DevOps & Cloud

Track ini membangun kemampuan mengelola infrastruktur, deployment, dan operasi sistem. Kamu akan belajar Linux, Docker, CI/CD, cloud, dan Kubernetes. Cocok untuk yang ingin jadi DevOps Engineer, Cloud Engineer, atau Site Reliability Engineer.

**Kompetensi per Level:**

- Beginner: 4 kompetensi
- Intermediate: 4 kompetensi
- Advanced: 3 kompetensi

## Level: Beginner

### Linux & Command-Line Fundamentals

- **Mastery Level:** Level 1 - Understand & Articulate
- **Evidence:** Penggunaan command line dasar untuk navigasi, manipulasi file, dan monitor resource
- **Exit Criteria:** Mengelola file server dan proses Linux tanpa GUI secara lancar

### Networking Fundamentals

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Konfigurasi DNS record, routing IP, dan SSL/TLS certificates
- **Exit Criteria:** Mengamankan koneksi domain dengan HTTPS secara benar

### Version Control Lanjutan (Git Branching & Workflow)

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Branching strategy tim yang rapi (Gitflow/Trunk-based)
- **Exit Criteria:** Mengatasi merge conflicts dan mengelola release branches secara terstruktur

### Containerization (Docker)

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** File Dockerfile dan docker-compose.yml untuk menjalankan aplikasi lokal
- **Exit Criteria:** Aplikasi berjalan dalam container terisolasi secara konsisten di komputer mana pun

## Level: Intermediate

### CI/CD Pipeline Design

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Pipeline CI/CD otomatis (GitHub Actions, GitLab CI)
- **Exit Criteria:** Menjalankan linters, tests, dan deployment otomatis secara berurutan

### Cloud Fundamentals (AWS / GCP / Azure)

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Konfigurasi server virtual (VM/EC2) dan serverless containers (Cloud Run)
- **Exit Criteria:** Infrastruktur berjalan dengan konfigurasi resource yang efisien secara biaya

### Infrastructure as Code (IaC)

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Script infrastruktur (Terraform/CloudFormation)
- **Exit Criteria:** Mereplikasi seluruh environment (staging ke production) otomatis tanpa klik manual

### Monitoring & Observability

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Setup monitoring (Prometheus, Grafana, ELK Stack)
- **Exit Criteria:** Alerting aktif untuk mendeteksi kegagalan server sebelum pengguna menyadarinya

## Level: Advanced

### Container Orchestration (Kubernetes)

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Manifest Kubernetes (Deployment, Service, Ingress)
- **Exit Criteria:** Cluster mengelola self-healing dan autoscaling container secara dinamis

### Security & Compliance (DevSecOps)

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Security scanning tools terintegrasi ke pipeline (Snyk, SonarQube)
- **Exit Criteria:** Memblokir build pipeline jika terdeteksi rahasia (secrets) atau celah keamanan bocor

### Disaster Recovery & SLA

- **Mastery Level:** Level 4 - Peer Instruction & Knowledge Share
- **Evidence:** Rencana pemulihan bencana (Disaster Recovery Plan) dan backup otomatis teruji
- **Exit Criteria:** Mengembalikan sistem aktif dari kegagalan total dalam batas waktu RTO/RPO
