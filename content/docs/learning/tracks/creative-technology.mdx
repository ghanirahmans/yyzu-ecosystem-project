# Creative Technology

Track ini membangun kemampuan menggabungkan kode dengan seni dan desain interaktif. Kamu akan belajar creative coding, motion design, generative art, dan physical computing. Cocok untuk yang ingin jadi Creative Technologist, Interactive Media Developer, atau Creative Director.

**Kompetensi per Level:**

- Beginner: 4 kompetensi
- Intermediate: 3 kompetensi
- Advanced: 3 kompetensi

## Level: Beginner

### Creative Coding Fundamentals

- **Mastery Level:** Level 1 - Understand & Articulate
- **Evidence:** Sketsa visual interaktif menggunakan kode (p5.js / Processing)
- **Exit Criteria:** Animasi berjalan dinamis merespons input mouse/keyboard

### Design Principles for Interactive Media

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Purwarupa interaktif dengan feedback visual yang natural
- **Exit Criteria:** Interaksi kreatif bermakna dan memandu eksplorasi pengguna

### Motion Design Fundamentals

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Storyboard animasi dengan kurva easing yang mulus
- **Exit Criteria:** Animasi memiliki timing natural (anticipation, squash, stretch)

### Tooling Ekosistem (After Effects / Rive)

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Animasi interaktif berbasis state machine (Rive)
- **Exit Criteria:** Animasi siap diekspor untuk kebutuhan runtime web/mobile

## Level: Intermediate

### Generative Design Systems

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Algoritma visual yang memproduksi variasi visual unik berdasarkan parameter dinamis
- **Exit Criteria:** Menghasilkan output unik tak terbatas dari satu set aturan kode

### Interactive Media & Real-Time Rendering

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Aplikasi web 3D interaktif menggunakan WebGL (Three.js)
- **Exit Criteria:** Render berjalan stabil minimal 60fps di browser desktop standar

### Web Animation & Lottie Integration

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Integrasi animasi Lottie ke antarmuka web/mobile
- **Exit Criteria:** Animasi dipicu oleh event scroll atau click tanpa degradasi performa

## Level: Advanced

### Data-Driven Visual Systems

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Visualisasi data seni generatif dinamis (generative data art)
- **Exit Criteria:** Mengubah dataset kompleks menjadi representasi visual artistik interaktif

### Physical Computing & Sensor Integration

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Instalasi interaktif menggunakan Arduino/Raspberry Pi dan sensor fisik
- **Exit Criteria:** Input dari dunia fisik memicu respons visual digital seketika

### Creative Direction & Concept Development

- **Mastery Level:** Level 4 - Peer Instruction & Knowledge Share
- **Evidence:** Pitch deck konsep kreatif proyek instalasi/media interaktif berskala besar
- **Exit Criteria:** Mengarahkan tim lintas disiplin (desainer, seniman, engineer) menuju satu visi karya
