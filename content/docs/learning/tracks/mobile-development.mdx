# Mobile Development

Track ini membangun kemampuan mengembangkan aplikasi mobile native dan cross-platform. Kamu akan belajar lifecycle platform, UI mobile, arsitektur MVVM, dan distribusi app store. Cocok untuk yang ingin jadi Mobile Developer, Android/iOS Developer, atau Flutter Developer.

**Kompetensi per Level:**

- Beginner: 5 kompetensi
- Intermediate: 4 kompetensi
- Advanced: 3 kompetensi

## Level: Beginner

### Platform Fundamentals & Lifecycle

- **Mastery Level:** Level 1 - Understand & Articulate
- **Evidence:** Aplikasi mobile sederhana yang mempertahankan state saat orientasi layar berubah
- **Exit Criteria:** Menjelaskan daur hidup (lifecycle) layar/activity saat background/foreground

### Mobile UI Components & Navigation Patterns

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Layout navigasi tab atau drawer sesuai panduan platform (HIG/Material)
- **Exit Criteria:** Alur perpindahan antar-layar mulus dan tidak menumpuk stack navigasi secara salah

### State Management Mobile

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Implementasi state local dan ViewModels
- **Exit Criteria:** Memisahkan logika tampilan dengan penyimpanan data sementara di memory

### Integrasi API & Offline-First Strategy

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Aplikasi mengambil data API dan menyimpannya di database lokal (Room/SQLite)
- **Exit Criteria:** Aplikasi dapat menampilkan data lama saat tidak ada jaringan internet

### Push Notification Architecture

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Integrasi push notification via Firebase (FCM) atau APNs
- **Exit Criteria:** Notifikasi masuk dan mengarahkan pengguna ke layar yang sesuai saat diklik

## Level: Intermediate

### Arsitektur Mobile (MVVM / Clean Architecture)

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Codebase mobile terbagi menjadi data, domain, dan presentation layer
- **Exit Criteria:** Logika bisnis diuji terpisah tanpa perlu menjalankan emulator/device

### Aksesibilitas Mobile (A11y)

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Aplikasi ramah screen reader (TalkBack/VoiceOver)
- **Exit Criteria:** Seluruh touch targets memiliki deskripsi semantik yang benar

### Performance & Memory Management

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Laporan profil memori bebas dari memory leak
- **Exit Criteria:** Tidak ada crash OutOfMemory saat memuat list gambar berukuran besar

### Mobile Testing (Unit & UI Testing)

- **Mastery Level:** Level 2 - Apply & Execute
- **Evidence:** Skenario UI testing terotomatisasi (Espresso/XCTest)
- **Exit Criteria:** Memverifikasi alur pendaftaran dan checkout berjalan lancar secara berkala

## Level: Advanced

### Platform API & Native Integration

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Aplikasi terintegrasi dengan hardware (Kamera, Biometrik, GPS)
- **Exit Criteria:** Mengelola perizinan (runtime permissions) dengan benar dan aman

### App Distribution & Release Management

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Upload berkas rilis (.apk/.ipa) ke Play Console atau App Store Connect
- **Exit Criteria:** Lolos proses peninjauan (review store) tanpa penolakan kebijakan dasar

### Cross-Platform Considerations

- **Mastery Level:** Level 3 - Evaluate & Architect
- **Evidence:** Memilih Flutter/React Native vs Native berdasarkan kendala performa dan resources
- **Exit Criteria:** Menulis ADR perbandingan performa engine
