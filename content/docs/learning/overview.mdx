# Learning Overview

Sistem pembelajaran di YYZU dirancang untuk membina talenta teknologi melalui kolaborasi dan praktik nyata. Melalui model ini, member berkembang dari tingkat dasar menuju kesiapan industri secara bertahap.

## Collaborative Learning

Member belajar bersama melalui diskusi, study group, mentoring, sharing session, dan peer learning.

Tujuannya bukan hanya memahami materi, tetapi membangun kebiasaan belajar yang aktif dan saling mendukung.
