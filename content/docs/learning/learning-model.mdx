# Model Pembelajaran (Learn • Build • Collaborate • Grow)

YYZU menggunakan pendekatan:
> **Learn -> Build -> Collaborate -> Grow**

## Learn
Member mempelajari fundamental, tools, mindset, dan workflow sesuai bidang yang diminati.

## Build
Member mulai menerapkan pengetahuan melalui mini project, challenge, prototype, atau task berbasis praktik.

## Collaborate
Member bekerja bersama dalam tim, berdiskusi, membagi peran, memberi feedback, dan menyelesaikan project secara kolektif.

## Grow
Member berkembang melalui pengalaman, evaluasi, kontribusi, mentoring, portfolio, dan exposure terhadap workflow yang lebih profesional.
