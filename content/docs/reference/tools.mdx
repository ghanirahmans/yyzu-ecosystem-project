Dokumen ini memberikan gambaran tools yang digunakan di YYZU. Kamu tidak perlu langsung menguasai semuanya. Mulai dari Discord dan Notion - dua tool yang akan kamu gunakan setiap hari.

## Panduan Setup Singkat

### Discord

1. Buat akun Discord jika belum punya
2. Terima undangan server YYZU (dikirim otomatis saat onboarding)
3. Atur nickname sesuai format: Nama - Track (contoh: Budi - Web Dev)
4. Baca channel #rules dan #announcements

### Notion

1. Buat akun Notion dengan email yang sama dengan onboarding
2. Terima undangan workspace YYZU
3. Bookmark halaman utama YYZU Ecosystem Project
4. Mulai dari halaman "Tentang YYZU" untuk memahami struktur dokumen

### GitHub

1. Buat akun GitHub jika belum punya
2. Terima invite ke organisasi GitHub YYZU
3. Atur SSH key atau Personal Access Token
4. Pastikan profile GitHub lengkap (foto, bio, link portfolio)

## Akses & Administrasi

### Siapa yang Mengelola Akses?

| Tool | Admin Utama | Backup Admin |
|---|---|---|
| Discord | Koordinator Umum | Divisi Media & Branding |
| Notion | Founder YYZU | Koordinator Umum |
| GitHub | Divisi PM | Founder YYZU |
| Figma | Divisi PM | Divisi Media & Branding |
| Google Drive | Koordinator Umum | Divisi SDM |

### Keamanan Akun

- Seluruh akun organisasi wajib menggunakan **Two-Factor Authentication (2FA)** sesuai AD Pasal 28.
- Password akun organisasi tidak boleh dibagikan melalui chat. Gunakan password manager.
- Akses dicabut otomatis saat keanggotaan berakhir sesuai ART Pasal 10.

## Troubleshooting FAQ

> **Tidak bisa akses Discord?** Pastikan kamu sudah menerima undangan dan akun Discord kamu aktif. Hubungi Divisi Media & Branding.

> **Notion error atau tidak bisa edit?** Coba refresh halaman. Jika masih bermasalah, cek apakah kamu sudah diundang ke workspace. Hubungi Koordinator Umum.

> **GitHub permission denied?** Pastikan kamu sudah accept invite ke organisasi YYZU dan SSH key sudah ter-setup. Hubungi Divisi PM.

> **Lupa password akun organisasi?** Jangan pernah share password di chat. Gunakan password manager. Hubungi admin tool terkait untuk reset.
