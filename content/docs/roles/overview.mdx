# Roles Overview

Struktur peran kepengurusan (roles) dirancang untuk memastikan kelancaran operasional dan koordinasi yang efektif di ekosistem YYZU.

## Pendahuluan
Dokumen ini disusun sebagai pedoman pembagian tanggung jawab, wewenang, ruang lingkup kerja, serta batasan setiap peran dalam ekosistem YYZU. Pembagian peran ini bertujuan untuk menciptakan koordinasi yang efektif, menghindari tumpang tindih pekerjaan, serta memastikan setiap fungsi ekosistem berjalan secara terarah, profesional, dan berkelanjutan.

Setiap pemegang peran bertanggung jawab terhadap bidang kerjanya masing-masing. Namun, seluruh peran tetap memiliki kewajiban untuk berkolaborasi secara aktif demi mendukung pencapaian visi dan tujuan ekosistem YYZU.

---

## Ringkasan Fokus Peran

| Peran | Fokus Utama |
| :--- | :--- |
| **Partnership** | Hubungan eksternal dan kerja sama ekosistem |
| **SDM Management** | Pengelolaan anggota dan budaya internal |
| **Event Organizer & Planner** | Perencanaan dan pelaksanaan kegiatan |
| **Product & Project Management** | Pengembangan produk, sistem, dan proyek |
| **Learning & Curriculum** | Pembelajaran dan peningkatan kompetensi anggota |
| **Media & Branding** | Citra ekosistem, publikasi, dan komunikasi publik |
| **Koordinator Umum BPH** | Koordinasi internal, penyelarasan program divisi, dan tata kelola organisasi |
| **Learners (Member)** | Pembelajaran aktif, kolaborasi mini projects, dan pengembangan portofolio |
| **Mentors (Volunteers)** | Bimbingan sukarela, review proyek, dan penyelarasan standar industri |
| **Mitra Kampus** | Sinergi kurikulum akademik, program magang MBKM, dan seminar mahasiswa |
| **Mitra Industri & Ekosistem** | Penyedia studi kasus riil, penyaluran karir (talent buyer), dan dukungan sumber daya |
