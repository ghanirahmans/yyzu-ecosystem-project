# Jenis Kegiatan Mentoring

Keterlibatan mentor dapat disesuaikan dengan waktu dan kapasitas masing-masing mentor. Secara umum, kontribusi mentor dapat masuk ke beberapa bentuk berikut:
1. Sesi arahan awal
2. Sesi validasi learning path
3. Sesi review rencana project
4. Sesi checkpoint project
5. Sesi review hasil project
6. Sesi sharing pengalaman industri
7. Sesi portfolio atau career review
8. Sesi Q&A teknis atau karier

Mentor tidak harus terlibat dalam semua kegiatan. Mentor dapat memilih bentuk kontribusi yang paling sesuai dengan bidang, pengalaman, dan ketersediaan waktunya.

## Peran Mentor Per Jenis Kegiatan
Kegiatan di YYZU dikelompokkan ke dalam beberapa kategori. Peran dan batasan mentor disesuaikan dengan karakteristik masing-masing kegiatan:

### 1. Learning Activity
Kegiatan belajar terstruktur (seperti study group, learning roadmap, basic challenge) untuk membantu member memahami fundamental dan tools dasar.
- **Peran Mentor:** Memberikan arahan materi utama, memvalidasi learning path, menjelaskan konsep penting, dan memberikan gambaran best practices di industri.
- **Batasan Mentor:** Tidak mengajar harian, tidak memantau progres harian member, dan tidak diwajibkan membuat materi ajar secara mandiri.
- **Batasan Keterlibatan:** 1-2 sesi pertemuan dalam satu siklus *Learning Sprint* (2 minggu).

### 2. Mentoring Activity
Sesi konsultasi khusus untuk membahas arah belajar, kendala teknis besar, keputusan arsitektur, dan bimbingan karir.
- **Peran Mentor:** Bertindak sebagai pembimbing khusus, membantu melihat kelemahan rencana belajar/proyek, dan memberikan saran prioritas pengembangan diri.
- **Batasan Mentor:** Bukan pengurus komunitas, tidak memimpin tim kerja, dan tidak diwajibkan selalu tersedia di chat pribadi.
- **Batasan Keterlibatan:** Hadir hanya pada sesi terjadwal (45-90 menit) atau checkpoint yang disepakati bersama.

### 3. Mini Project (Skala Kecil)
Project latihan dasar 6 minggu yang dikerjakan berkelompok untuk menguji pemahaman fundamental member.
- **Peran Mentor:**
  - Membimbing tim di fase belajar awal (1-2 sesi).
  - Meninjau ide dan scope project agar tidak terlalu rumit (1 sesi).
  - Memberikan arahan jika ada hambatan teknis yang besar (blocker).
  - Melakukan review hasil project akhir dan memberikan masukan portofolio (1 sesi).
- **Batasan Mentor:** Tidak memantau progres harian kelompok (tugas Project Leader), tidak ikut menulis kode, dan tidak mengambil keputusan operasional harian tim.
- **Batasan Keterlibatan:** **4-5 sesi dalam 6 minggu** (mengikuti pola Level 2 - Cycle Mentor).

### 4. Collaborative Project (Skala Menengah)
Project tim lintas disiplin (8-12 minggu) yang menerapkan workflow industri secara profesional.
- **Peran Mentor:** Memvalidasi rencana arsitektur sistem, memberikan masukan standar penulisan kode/desain, serta bertindak sebagai penguji (*client/instructor*) saat demo.
- **Batasan Mentor:** Tidak membagi tugas kelompok, tidak mengejar deadline anggota kelompok, dan tidak ikut menelusuri/memperbaiki bug sistem secara mikro.
- **Batasan Keterlibatan:** **3-5 sesi dalam 8-12 minggu** (mengikuti pola Level 3 - Project Advisor).

### 5. Internal Project (Aplikasi internal YYZU)
Pengembangan platform internal YYZU secara modular (2-3 bulan per fase) yang dikerjakan oleh member terpilih.
- **Peran Mentor:** Memberikan arahan teknis jangka panjang (*maintainability*), meninjau arsitektur database/cloud, serta meninjau kelayakan kode sebelum produk rilis.
- **Batasan Mentor:** Bukan maintainer kode harian, bukan product owner yang menetapkan timeline, dan tidak mengurusi deployment server.
- **Batasan Keterlibatan:** **Milestone-based review (3-4 sesi per fase pengerjaan)**.

### 6. Review Activity
Kegiatan evaluasi portofolio, draf desain, kode program, atau simulasi presentasi.
- **Peran Mentor:** Memberikan umpan balik konstruktif terhadap kualitas teknis, kegunaan UI/UX, dan kesiapan berkas lamaran kerja/portofolio.
- **Batasan Mentor:** Tidak menulis ulang kode/desain member dan tidak melakukan peninjauan di luar batas waktu yang telah dijadwalkan (45-90 menit per sesi).

### 7. Industry & Career Activity
Webinar industri, CV review, portofolio review, serta simulasi wawancara (*mock interview*).
- **Peran Mentor:** Berbagi pengalaman kerja, memberikan tips wawancara, dan menyelaraskan pemahaman member dengan ekspektasi nyata di industri.
- **Batasan Mentor:** Mentor tidak menjamin kelulusan kerja, tidak wajib memberikan surat rekomendasi/referral otomatis, dan tidak bertindak sebagai agen pencari kerja.

### 8. Ecosystem & Collaboration Activity
Aktivitas internal komunitas (weekly sync, sharing antar member, showcase internal).
- **Peran Mentor:** Bersifat opsional. Mentor dapat hadir sesekali sebagai tamu undangan (*guest*) untuk memberikan wawasan tambahan.
- **Batasan Mentor:** Tidak ikut mengelola kepengurusan komunitas, tidak menjadi admin grup, dan tidak bertanggung jawab atas keaktifan harian komunitas.
