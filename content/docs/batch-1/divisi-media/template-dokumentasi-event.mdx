> Divisi Media & Branding | Batch 1: Economic & Education

> Total Event: 32 event

---

## 1. Checklist Dokumentasi per Event

### Sebelum Event
- [ ] Siapkan perangkat foto/video (kamera/HP, tripod, pencahayaan)
- [ ] Pastikan baterai penuh dan storage cukup
- [ ] Cek jadwal event: waktu, platform, pembicara
- [ ] Siapkan template notulensi
- [ ] Koordinasi dengan PIC event untuk rundown
- [ ] Siapkan backdrop/banner YYZU jika offline

### Selama Event
- [ ] Ambil foto pembukaan (1-2 foto)
- [ ] Screenshot tampilan awal (untuk event online)
- [ ] Foto/screenshot pembicara saat presentasi
- [ ] Foto/screenshot peserta (grid view untuk online)
- [ ] Foto/screenshot momen interaksi (Q&A, diskusi)
- [ ] Catat notulensi: ringkasan, keputusan, action items
- [ ] Ambil foto/screenshot slide penting
- [ ] Rekam video singkat untuk recap (15-30 detik)

### Setelah Event
- [ ] Sortir dan pilih foto terbaik (minimal 5 foto)
- [ ] Screenshot momen penting dari rekaman (jika ada)
- [ ] Selesaikan notulensi dalam 24 jam
- [ ] Upload ke folder arsip sesuai naming convention
- [ ] Siapkan draft konten untuk sosmed (jika perlu)
- [ ] Kirim dokumentasi ke Koordinator Divisi

## 2. Guidelines Foto/Screenshot yang Baik

### Foto (untuk Event Offline)

**Komposisi:**
- Gunakan rule of thirds
- Pastikan subjek utama terlihat jelas
- Hindari blur, noise, dan pencahayaan buruk
- Ambil dari berbagai angle: wide shot, medium shot, close-up

**Yang harus difoto:**
- Pembicara saat presentasi
- Peserta saat mendengarkan/diskusi
- Suasana ruangan secara keseluruhan
- Momen interaksi (tanya jawab, kerja kelompok)
- Detail: name tag, materi, makanan/snack
- Banner/backdrop YYZU

**Tips teknis:**
- Gunakan mode burst untuk momen gerak
- ISO rendah untuk indoor (noise minimal)
- White balance sesuai pencahayaan ruangan
- Minimal 5 foto per event, idealnya 10-15

### Screenshot (untuk Event Online)

**Yang harus di-screenshot:**
- Tampilan awal dengan daftar peserta
- Pembicara saat sedang presentasi
- Slide presentasi yang penting
- Grid view peserta (untuk bukti kehadiran)
- Momen Q&A atau diskusi
- Poll/quiz jika ada

**Tips teknis:**
- Gunakan resolusi layar penuh
- Crop bagian yang tidak relevan (toolbar, tab browser)
- Pastikan nama peserta terlihat
- Simpan dalam format PNG untuk kualitas terbaik

### Video (untuk Story/Reels)

**Spesifikasi:**
- Durasi: 15-30 detik per klip
- Orientasi: portrait (9:16) untuk story/reels
- Resolusi: minimal 1080p
- Stabil: gunakan tripod atau permukaan datar

**Yang harus direkam:**
- Opening event (10-15 detik)
- Momen penting (15-30 detik)
- Closing/penutup (10 detik)
- Reaksi peserta (opsional)

## 4. Template Video Recap Singkat (15-30 detik)

### Struktur Video Story/Reels

**Detik 0-3: Opening**
- Text overlay: "[Nama Event] | YYZU Batch 1"
- Background: foto/screenshot event
- Music: upbeat, mulai pelan

**Detik 3-8: Suasana Event**
- Klip: tampilan peserta, pembicara, atau ruangan
- Text overlay: "[Topik Event]"
- Transisi: smooth cut atau swipe

**Detik 8-18: Highlights**
- Klip: momen penting, slide presentasi, interaksi
- Text overlay: poin-poin utama (1-2 kata per poin)
- Transisi: quick cuts setiap 3-4 detik

**Detik 18-25: Momen Menarik**
- Klip: Q&A, reaksi peserta, atau momen lucu
- Text overlay: quote dari pembicara/peserta
- Music: naik sedikit energinya

**Detik 25-30: Closing**
- Text overlay: "Terima kasih! Sampai jumpa di event berikutnya 🙌"
- Logo YYZU
- Music: fade out

### Spesifikasi Teknis
- Durasi: 15-30 detik (idealnya 20-25 detik)
- Orientasi: portrait 9:16 (1080x1920)
- Format: MP4 (H.264)
- Frame rate: 30fps
- Audio: musik background, volume 30-40% dari klip asli

## 6. Naming Convention File Dokumentasi

### Foto

```
Format: [YYYY-MM-DD]_[nama-event]_[urutan]_[deskripsi].jpg

Contoh:
2026-07-14_orientasi-batch1_01_pembukaan.jpg
2026-07-14_orientasi-batch1_02_pembicara.jpg
2026-07-14_orientasi-batch1_03_peserta-grid.jpg
2026-07-14_orientasi-batch1_04_sesi-tanya-jawab.jpg
```

### Screenshot

```
Format: [YYYY-MM-DD]_[nama-event]_[urutan]_[deskripsi].png

Contoh:
2026-07-14_orientasi-batch1_01_tampilan-awal.png
2026-07-14_orientasi-batch1_02_slide-program.png
2026-07-14_orientasi-batch1_03_peserta-grid.png
```

### Video

```
Format: [YYYY-MM-DD]_[nama-event]_[tipe]_[durasi].mp4

Tipe: recap, highlight, full, clip
Durasi: 15detik, 30detik, 1menit, full

Contoh:
2026-07-14_orientasi-batch1_recap_15detik.mp4
2026-07-14_orientasi-batch1_highlight_30detik.mp4
2026-07-14_orientasi-batch1_full.mp4
```

### Notulensi

```
Format: [YYYY-MM-DD]_[nama-event]_notulensi.md

Contoh:
2026-07-14_orientasi-batch1_notulensi.md
```

### Metadata

```
Format: [YYYY-MM-DD]_[nama-event]_metadata.txt

Isi metadata.txt:
----------------------------------------
Event: [nama event]
Tanggal: [dd MMMM yyyy]
Waktu: [HH:MM - HH:MM] WIB
Platform: [platform]
Pembicara: [nama]
Peserta: [jumlah]
Fotografer: [nama]
Jumlah foto: [angka]
Jumlah screenshot: [angka]
Video: [ada/tidak]
Notulensi: [selesai/belum]
Upload date: [dd MMMM yyyy]
----------------------------------------
```

### Aturan Penamaan Umum

1. **Gunakan lowercase** untuk semua nama file
2. **Ganti spasi dengan strip (-)** bukan underscore (_) untuk keterbacaan
3. **Tanggal selalu di depan** untuk sorting kronologis
4. **Urutan menggunakan 2 digit**: 01, 02, 03 (bukan 1, 2, 3)
5. **Deskripsi singkat tapi jelas**: maksimal 3-4 kata
6. **Ekstensi file sesuai format**: .jpg, .png, .mp4, .md

---

## Catatan untuk PIC Dokumentasi

1. **Kualitas lebih penting dari kuantitas.** 5 foto bagus lebih baik dari 20 foto blur.
2. **Jangan lupa screenshot.** Untuk event online, screenshot adalah bukti utama.
3. **Notulensi selesai dalam 24 jam.** Jangan tunggu sampai lupa detailnya.
4. **Backup semua file.** Simpan di minimal 2 tempat (local + cloud).
5. **Perhatikan privasi.** Jangan publish foto tanpa izin, terutama untuk konten publik.
6. **Komunikasi dengan tim.** Koordinasi dengan Koordinator Divisi sebelum dan sesudah event.
