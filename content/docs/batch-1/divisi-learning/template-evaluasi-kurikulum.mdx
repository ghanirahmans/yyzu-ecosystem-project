# Template Evaluasi Kurikulum Batch 1

Template survei dan rekap evaluasi efektivitas kurikulum YYZU Batch 1. Digunakan setelah batch selesai untuk mengevaluasi materi dan mentor, serta menyusun rekomendasi perbaikan untuk Batch 2.

---

## Survei Efektivitas Materi (Untuk Talenta)

Instruksi: Berikan rating 1-5 untuk setiap pertanyaan.
- 1 = Sangat Tidak Setuju / Tidak Membantu

- 3 = Netral
- 4 = Setuju
- 5 = Sangat Setuju / Sangat Membantu

| No | Pertanyaan | Rating (1-5) |
|----|------------|:------------:|
| 1 | Apakah materi Learning Sprint cukup untuk memulai project? | [ ] |

| 3 | Apakah Workshop #2 (Teknis Track) relevan dengan track yang kamu pilih? | [ ] |
| 4 | Apakah materi Workshop #3 (Persiapan Collab) cukup untuk memulai project kolaborasi? | [ ] |
| 5 | Apakah Workshop #4 (Lanjutan) menambah pengetahuan teknis kamu? | [ ] |
| 6 | Apakah Workshop #5 (Teknis Track 2) relevan dengan kebutuhan track kamu? | [ ] |
| 7 | Apakah Workshop #6 (Persiapan Presentasi) berguna untuk mempersiapkan final presentation? | [ ] |
| 8 | Apakah study group efektif untuk belajar dan berdiskusi? | [ ] |

## Instruksi Pengisian untuk Koordinator

1. **Distribusikan survei** ke semua talenta di minggu terakhir batch (W17-W18)
2. **Beri tenggat waktu** minimal 5 hari untuk mengisi survei
3. **Analisis hasil** menggunakan tabel rekap di atas
4. **Susun laporan evaluasi** menggunakan template yang tersedia
5. **Presentasikan rekomendasi** ke divisi learning dan manajemen YYZU
6. **Arsipkan** semua data evaluasi untuk referensi Batch 2

Tips:
- Pastikan survei anonim agar talenta bisa memberikan feedback jujur
- Kumpulkan evaluasi mentor secara terpisah dari evaluasi materi
- Gunakan data kuantitatif (rating) dan kualitatif (terbuka) secara berdampingan
- Identifikasi pola, bukan hanya rata-rata - lihat distribusi rating juga
