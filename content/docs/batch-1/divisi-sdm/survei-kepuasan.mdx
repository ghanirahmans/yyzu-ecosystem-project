# Survei Kepuasan YYZU Batch 1

Survei ini dilakukan di akhir batch (W17, Feedback Session) untuk mengukur kepuasan talenta dan mendapat masukan untuk perbaikan batch selanjutnya.

---

## Instruksi Pengisian

- Survei ini ANONIM. Nama tidak akan tercatat.
- Jawab dengan jujur. Masukanmu sangat berharga untuk perbaikan YYZU.
- Isi semua pertanyaan yang wajib (*).
- Waktu pengisian: 5-10 menit.

## Catatan Pelaksanaan Survei

### Waktu Distribusi
- Survei dibagikan saat Feedback Session (W17)
- Deadline pengisian: 3 hari setelah Feedback Session
- Reminder dikirim di hari ke-2 jika belum semua mengisi

### Target
- Minimal 80% talenta mengisi survei
- Rating keseluruhan rata-rata minimal 4/5

### Proses Rekap
1. Kumpulkan semua jawaban dari Google Form
2. Buat rekap menggunakan template di atas
3. Analisis insight dari open-ended
4. Susun laporan untuk Koordinator Umum
5. Presentasikan di rapat evaluasi batch
