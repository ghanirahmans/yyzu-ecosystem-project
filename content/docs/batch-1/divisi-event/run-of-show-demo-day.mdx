## 5. Backup Plan

### Masalah Teknis

| Masalah | Solusi | PIC |
|---------|--------|-----|
| Link Zoom/Meet tidak bisa diakses | Gunakan link backup yang sudah disiapkan di grup WA | Koordinator Teknis |
| Screen share tidak berfungsi | Minta presenter kirim file ke host, host yang share | Koordinator Teknis |
| Audio presenter tidak terdengar | Minta mute/unmute, cek mic, switch ke HP jika perlu | Runner |
| Koneksi internet presenter putus | Skip ke tim berikutnya, presentasi yang putus dijadwal ulang di akhir | Ketua Event |
| Recording gagal | Gunakan backup recording (OBS di device lain) | Koordinator Teknis |
| Platform down total | Pindah ke Google Meet/Zoom backup yang sudah disiapkan | Koordinator Teknis |

### Tim Tidak Hadir

| Kondisi | Solusi | PIC |
|---------|--------|-----|
| 1 orang tim tidak hadir | Tim tetap presentasi, bagi porsi yang hadir | Ketua Event |
| Mayoritas tim tidak hadir | Tukar urutan dengan tim berikutnya, tim yang tidak hadir presentasi di akhir jika memungkinkan | Ketua Event |
| Seluruh tim tidak hadir | Skip ke tim berikutnya, jadwalkan presentasi terpisah di lain hari | Ketua Event |
| Assessor tidak hadir | Minta assessor kirim penilaian tertulis, tetap lanjutkan presentasi | PIC Narasumber |

### Masalah Lain

| Masalah | Solusi | PIC |
|---------|--------|-----|
| Melebihi waktu | MC mengingatkan waktu, skip Q&A jika perlu, potong break | MC |
| Peserta terlalu sedikit | Tetap lanjutkan, record untuk yang tidak hadir | Ketua Event |
| Chat tidak sopan | Moderator mute/warning, arahkan ke topik | Moderator Q&A |
