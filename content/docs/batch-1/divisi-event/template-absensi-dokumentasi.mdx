```
# Foto/Screenshot Event: [Judul Event]
# Tanggal: [DD MMMM YYYY]

1. Ambil minimal 3 screenshot per event:
   - Screenshot awal sesi (saat semua peserta sudah masuk)
   - Screenshot saat sesi berlangsung (narasumber sedang presentasi)
   - Screenshot akhir sesi (sebelum closing)

2. Pastikan terlihat:
   - Nama event di title bar/header
   - Jumlah peserta
   - Tanggal dan waktu

3. Format:
   - Simpan sebagai PNG atau JPG
   - Simpan di folder: /YYZU-Batch1/Dokumentasi/[Tanggal]-[NamaEvent]/
```

## Catatan Penggunaan

1. **Absensi per event**: Isi langsung saat event berlangsung. PIC Absensi bertanggung jawab.
2. **Rekap kehadiran per talenta**: Update setiap selesai event, jangan ditumpuk.
3. **Notulensi**: Dikerjakan oleh notulis yang ditunjuk, selesai maksimal H+1.
4. **Screenshot**: Ambil saat event, jangan di-skip. Simpan langsung ke folder yang benar.
5. **Feedback**: Kumpulkan segera setelah event. Bisa via Google Form atau langsung di chat.
6. **Rekap dokumentasi**: Update setiap selesai event. Jangan sampai ada event tanpa dokumentasi.
